## Steps to Reproduce:

### Backend:
```bash
git clone https://github.com/rawataman101/amcodr-project-management-app
```
```bash
cd amcodr-project-management-app
cd backend
```
```bash
python -m venv venv
source venv/bin/activate
```
```bash
pip install -r requirements.txt
```

### Frontend:

```bash
git clone https://github.com/rawataman101/amcodr-project-management-app
```
```bash
cd amcodr-project-management-app
cd frontend
```
```bash
npm install 
npm run dev
```
